# 鹿野的日常（Luye Daily）

> 罗小黑战记同人 · 桌面伴侣 App 原型

---

## 产品定位

「鹿野的日常」是一款**角色扮演型桌面伴侣**应用。

鹿野是《罗小黑战记》中的**妖灵会馆感知组执行者**，不是宠物，不是助手，而是与用户平等的伙伴。他傲娇、毒舌，但内心关心你。

**核心差异**：不是 AI 助手，不是聊天机器人，而是一个有自己人格、情绪和日程的**角色伴侣**。

---

## 技术栈

| 层 | 选型 |
|---|--------|
| 前端 | 单文件 HTML/CSS/JS（原型），无框架依赖 |
| 对话 | DeepSeek API（可在设置页配置自定义端点） |
| 状态持久化 | localStorage（六维属性、羁绊值、每日状态） |
| 换装系统 | 25 套造型，CSS 姿势切换 |
| PWA | manifest.json + service-worker.js，支持安装到手机桌面 |
| 离线降级 | API 不可用时自动切换本地对话池 |

## 角色设定

**鹿野** —— 《罗小黑战记》妖灵会馆感知组执行者。

| 属性 | 说明 |
|------|------|
| 身份 | 妖灵会馆感知组执行者（不是猫妖宠物） |
| 性格 | 傲娇、毒舌、内敛但关心人 |
| 说话风格 | 句首不带"……"，不卖萌，不"喵喵叫" |
| 与用户关系 | 平等的战友/同门，不是主仆 |

**绝对禁止**（SYSTEM_PROMPT 规则）：
- 禁止自称"AI"/"助手"/"机器人"/"模型"
- 禁止说"作为AI"/"我不能"/"我无法"
- 禁止卖萌语气（"喵"/"主人"等）
- 每次输出 1~3 句，每句 ≤ 25 字

## 六维属性系统

| 属性 | 说明 | 影响 |
|------|------|------|
| 灵力 | 修行根基，执行者核心能力 | 对话语气、升级台词 |
| 心情 | happy / normal / low / angry 四档 | 对话池选择 |
| 羁绊 | 0~100，与用户亲密度 | 影响对话池风格 |
| 精力 | 0~100，随时间衰减 | 影响互动反馈 |
| 饱腹 | 0~100，需喂食维持 | 低值时触发抱怨台词 |
| 清洁 | 0~100，需梳洗维持 | 影响修行效率台词 |

所有属性通过 `localStorage` 持久化，关闭再开保留状态。

## PWA 支持

已配置 `manifest.json` + `service-worker.js`：

- 手机浏览器访问后可 **"添加到主屏幕"**
- 独立运行（无浏览器地址栏）
- 离线缓存核心资源
- 图标：`icons/icon-192.png` / `icon-512.png`

**部署后访问地址**：  
https://abner09.github.io/luye-companion/

## 本地运行

无需构建，直接打开即可：

```bash
# 方式一：直接双击 luye-companion.html（部分功能受限）
# 方式二：本地 HTTP 服务（推荐）
npx serve .
# 然后访问 http://localhost:3000/luye-companion.html
```

> Service Worker 只能在 `http://` 或 `https://` 下运行，  
> 直接 `file://` 打开无法激活 PWA 功能。
